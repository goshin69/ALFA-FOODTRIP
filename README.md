## 📖 Descripción

**Koalicius** es una plataforma web interactiva orientada a personas apasionadas por la gastronomía. Su objetivo es ofrecer un espacio digital donde los usuarios puedan **descubrir, publicar, compartir y organizar recetas culinarias** de manera sencilla y atractiva.

La plataforma ha sido diseñada con un enfoque centrado en la experiencia del usuario, incorporando una interfaz moderna, intuitiva y visualmente atractiva. Gracias a ello, los usuarios pueden explorar distintas recetas, visualizar contenido multimedia y participar activamente dentro de una comunidad gastronómica.

Además, Koalicius promueve la interacción social mediante funcionalidades como publicaciones, comentarios, valoraciones y seguimiento de perfiles, convirtiéndose no solo en un repositorio de recetas, sino en un espacio colaborativo de aprendizaje e inspiración culinaria.

## 🚀 Características de la Plataforma

### 📝 Publicación de Recetas
Permite a los usuarios crear y compartir recetas incluyendo:
- Título
- Descripción
- Ingredientes
- Procedimiento

### 🎥 Subida de Videos Integrada
Los usuarios pueden adjuntar videos directamente desde su dispositivo sin necesidad de enlaces externos.

### 🖼️ Contenido Multimedia
Cada receta puede incluir:
- Imágenes
- Videos
- Etiquetas culinarias

### 👤 Perfiles de Usuario
Cada usuario dispone de un perfil personalizado para:
- Publicar recetas
- Gestionar contenido
- Visualizar estadísticas
- Interactuar con otros usuarios

### 🔍 Buscador Avanzado
Sistema de búsqueda por:
- Nombre
- Ingredientes
- Etiquetas

### 🌎 Comunidad Interactiva
La plataforma incorpora:
- Comentarios
- Valoraciones
- Seguimiento de perfiles

## ⚙️ Cómo usar la plataforma

### 1️⃣ Acceder a la plataforma
Ingresa desde tu navegador web.

### 2️⃣ Registrarse o iniciar sesión
Crea una cuenta o inicia sesión para acceder a todas las funcionalidades.

### 3️⃣ Explorar recetas
Utiliza el buscador o navega por categorías para descubrir nuevas recetas.

### 4️⃣ Visualizar contenido
Consulta recetas detalladas con:
- Ingredientes
- Procedimientos
- Imágenes
- Videos

### 5️⃣ Publicar recetas
Completa el formulario de creación con:
- Título
- Ingredientes
- Descripción
- Procedimiento

### 6️⃣ Subir archivos multimedia
Adjunta imágenes y videos desde tu dispositivo para enriquecer cada publicación.

### 7️⃣ Interactuar con la comunidad
- Comenta recetas
- Califica publicaciones
- Sigue a otros usuarios

## 🚀 Instalación local

Sigue estos pasos para ejecutar el proyecto en tu entorno local:

1. Clona este repositorio:
   ```bash
   git clone https://github.com/goshin69/ALFA-FOODTRIP.git
   ```

2. Copia la carpeta del proyecto en el directorio `htdocs` de XAMPP.

3. Inicia los servicios **Apache** y **MySQL** desde el panel de control de XAMPP.

4. Importa la base de datos en **phpMyAdmin** si el proyecto incluye un archivo `.sql`.

5. Abre tu navegador y entra a:
   ```text
   http://localhost/ALFA-FOODTRIP
   ```
